# Fitness Core - Frontend

This is the frontend for Fitness Core, a Progressive Web App (PWA) built using React, Vite, and TypeScript.

## Getting Started

### Local Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy `.env.template` to `.env` and configure your API URL.
3. Start the dev server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build
To generate the production build with the PWA Service Worker:
```bash
npm run build
```
The compiled files will be located in the `dist/` directory.
